Command Line Prediction API Sample Instructions

A starter application has been generated for you.

  1. Install Java Platform (JDK 6 or higher).
  2. Install Maven (3.1.0 or higher). You may need to set your JAVA_HOME.
  3. Download the starter application and unzip it.
  4. Download the client secrets file and overwrite the src/main/resources/client_secrets.json file that was included in the application.
  5. Open a terminal, change to the prediction-cmd-line-sample directory you just unzipped, and execute: 
    mvn compile
mvn -q exec:java
    This will compile and run your starter application in the command line.

Congratulations, you are up and running! At this point you will want to add calls into the Prediction API to the /src/main/java/com/google/api/services/samples/prediction/cmdline/PredictionSample.java file. Please read the file carefully, it contains detailed information in the comments.

To enable logging of HTTP requests and responses (highly recommended when debugging), please take a look at the logging.properties file.

For more information on the Prediction API Java library surface you can visit:

https://develpers.google.com/resources/api-libraries/documentation/prediction/v1.6/java/latest/

Also check out the Java Client Library documentation, and get more information on the Prediction API at:

https://developers.google.com/prediction/docs/developer-guide

Setup Project in Eclipse

Prerequisites: install Eclipse, and the Maven plugin.

  • Setup Eclipse Preferences
    • Window > Preferences... (or on Mac, Eclipse > Preferences...)
    • Select Maven
      • check on "Download Artifact Sources"
      • check on "Download Artifact JavaDoc"
  • Import prediction-cmd-line-sample project
    • File > Import...
    • Select "General > Existing Project into Workspace" and click "Next"
    • Click "Browse" next to "Select root directory", find prediction-cmdline-sample and click "Next"
    • Click "Finish"
  • Run
    • Right-click on project prediction-cmd-line-sample
    • Run As > Java Application
    • If asked, type "PredictionSample" and click OK
